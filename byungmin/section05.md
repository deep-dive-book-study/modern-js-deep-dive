## 5장 표현식과 문

### 표현식과 문 (Expression and Statement)
1. 기본 개념 정의
- 값 (Value): 식이 평가(Evaluation)되어 생성된 결과
- 리터럴 (Literal): 사람이 이해할 수 있는 문자나 기호를 사용해 값을 생성하는 표기법(예: 10, 'Hello', true, [], {})

2. 표현식 (Expression)
- 정의: 값으로 평가될 수 있는 모든 문. 표현식이 실행되면 새로운 값을 생성하거나 기존 값을 참조함 (결과적으로 하나의 값이 남는다.)
- 특징: 표현식은 그 자체로 값이 될 수 있다. 따라서 값에 올 수 있는 모든 자리에 표현식도 올 수 있다.

```javascript
100              // 리터럴 표현식
50 + 50          // 연산자 표현식
score            // 식별자 참조 표현식 (변수에 값이 있다는 가정)
getScore()       // 함수 호출 표현식
```
3. 문 (Statement)
- 정의: 프로그램을 구성하는 기본 단위이자 최소 실행 단위(컴퓨터에게 내리는 명령)
- 구성: 여러 토큰(키워드, 식별자, 연산자 등)으로 구성됨
- 종류: 선언문, 할당문, 조건문, 반복문 등

```javascript
var x;           // 변수 선언문
x = 5;           // 할당문 (표현식이기도 함)
if (x > 0) { }   // 조건문
for (i=0; i<5; i++) { } // 반복문
```
