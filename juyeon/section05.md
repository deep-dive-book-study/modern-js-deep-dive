### 5장 표현식과 문

#### 값

식(표현식)이 평가되어 생성된 결과

```js
var sum = 10 + 20;
```

sum에 할당되는 것은 결과값인 **30**

#### 리터럴

사람이 이해할 수 있는 문자 또는 약속한 기호를 사용해 값을 생성하는 표기법

- 정수 리터럴
- 부동소수점 리터럴
- 2진수 리터럴(0b로 시작)
- 8진수 리터럴(0o로 시작)
- 16진수 리터럴(0x로 시작)
- 문자열 리터럴
- 불리언 리터럴
- null 리터럴
- undefined 리터럴
- 객체 리터럴
- 배열 리터럴
- 함수 리터럴
- 정규 표현식 리터럴

#### 표현식

값으로 평가될 수 있는 문(변수에 할당 가능 한 문)

#### 문

프로그램을 구성하는 기본 단위이자 최소 실행 단위
여러 토큰(문법적인 의미를 가지며, 문법적으로 더이상 나눙 수 없는 코드의 기본 요소)

#### 세미콜론과 세미콜론 자동 삽입기능

자바스크립트 엔진은 소스코드를 해석할 때 문의 끝이라고 예측되는 지점에 세미콜론을 자동으로 붙여줌
그러나 세미콜론 사용을 권장

#### 표현식과 표현식이 아닌 문

문은 2가지로 구분됨

- 표현식: 변수에 할당 가능(x + 100, x=100)
- 표현식이 아닌 문: 변수에 할당 불가능 (var x)
