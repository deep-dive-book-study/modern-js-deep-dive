### 15장 let, const 키워드와 블록 레벨 스코프

#### var 키워드

1. 중복 선언 가능
2. 함수 레벨 스코프
3. 변수 호이스팅이 될 때 선언 단계와 초기화 단계가 한번에 진행됨, 선언 이전에 참조시 에러를 발생시키지 않음
4. 전역 변수, 전역 함수, 선언하지 않은 변수에 값을 할당한 암묵적 전역은 전역 객체 window의 프로퍼티가 됨

#### let 키워드, const 키워드

1. 변수 중복 선언 금지
2. 블록 레벨 스코프
3. 변수 호이스팅이 될 때 선언 단계가 진행되고, 초기화 단계는 변수 선언문에 도달했을 때 실행됨, 이때 스코프의 시작 지점부터 초기화 단계 시작 지점까지 변수를 참조할 수 없는 일시적 사각 지대(TDZ)에 들어감, 선언 이전에 참조시 에러 발생
4. 전역 변수는 전역 객체 window의 프로퍼티가 아님(보이지 않는 개념적인 블록 내에 존재)

#### const 키워드가 let 키워드와 다른 점

1. 선언과 동시에 반드시 초기화
2. 재할당 금지(객체의 값은 변경 가능)
