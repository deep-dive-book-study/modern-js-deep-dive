### 데이터 타입

- 원시 타입
  - `Number`
    - 정수만을 위한 타입이 없고 모든 수를 실수로 처리
    - `Infinity`
    - `-Infinity`
    - `NaN`
      - 자신과 일치하지 않는 유일한 값
      - Not a Number
      - 값이 유효한 숫자가 아니라는 것을 의미
  - `String`
  - `Boolean`
  - `undefined`
    - 개발자가 의도적으로 할당하기 위한 값이 아닌 자바스크립트 엔진이 변수를 초기화할 때 사용하는 값
    - `undeclared`
      - 자바스크립트의 값이 아님
      - 접근 가능한 스코프 내에 변수 자체가 선언조차도 되어있지 않은 상태를 의미
  - `null`
    - 변수에 값이 없다는 것을 의도적으로 명시할 때 사용하는 값
  - `Symbol`
- 객체 타입
  - 객체, 함수, 배열

---

### 심벌 타입

- 변경 불가능한 원시 타입의 값
- 다른 값과 중복되지 않는 유일무이한 값
- `Symbol` 함수를 호출해 생성하며 이때 생성된 심벌 값은 외부에 노출되지 않고, 다른 값과 절대 중복 X

---

### 데이터 타입의 필요성

- 값을 저장할 때 확보해야 하는 메모리 공간의 크기를 결정하기 위해
- 값을 참조할 때 한 번에 읽어 들여야 할 메모리 공간의 크기를 결정하기 위해
- 메모리에서 읽어 들인 2진수를 어떻게 해석할지 결정하기 위해

---

### 동적 타이핑

- 자바스크립트의 변수는 선언이 아닌 할당에 의해 타입이 결정(타입 추론)이 되며, 재할당에 의해 변수의 타입은 언제든지 동적으로 변할 수 있는데 이러한 특징을 동적 타이핑이라 함
- 자바스크립트, 파이썬, PHP, 루비, 리스프, 펄 등
- 유연성은 높지만 신뢰성은 낮음
